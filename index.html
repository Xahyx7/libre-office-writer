Sidebar & Styles in LibreOffice Writer

What are Styles?

Styles in LibreOffice Writer are predefined formatting settings that help maintain consistency throughout documents. Instead of manually formatting each element, styles allow you to apply consistent formatting with just one click.

üí° Benefits of Using Styles

  • Consistency: Ensures uniform formatting throughout the document
  • Time-saving: Apply complex formatting instantly
  • Easy updates: Change formatting globally by modifying the style
  • Professional appearance: Creates well-structured documents

Accessing the Styles Sidebar

The Styles sidebar is the primary tool for managing styles in LibreOffice Writer. You can access it through multiple methods:

Method 1: Function Key

Press F11 key (‚åò+T on macOS)

Method 2: Menu Bar

Go to View > Styles

Method 3: Sidebar

Click on the Styles icon in sidebar

Method 4: Right-click

Right-click and select Styles

Six Categories of Styles

üìÑ 1. Page Styles

Purpose: Controls page layout, margins, headers, footers, and page orientation

Examples: Default page, First page, Left page, Right page

Usage: Different page layouts within the same document

üìù 2. Paragraph Styles

Purpose: Formats entire paragraphs including font, spacing, indentation, and borders

Examples: Heading 1-10, Text Body, Caption, Title

Shortcuts: Ctrl+0 (Text Body), Ctrl+1 (Heading 1), Ctrl+2 (Heading 2)

üî§ 3. Character Styles

Purpose: Formats selected text within paragraphs

Examples: Emphasis, Strong Emphasis, Source Text

Application: Apply to selected words or phrases only

üñºÔ∏è 4. Frame Styles

Purpose: Formats text boxes, images, and other objects

Examples: Formula, Graphics, OLE objects

Usage: Consistent formatting for inserted objects

üìã 5. List Styles

Purpose: Formats numbered and bulleted lists

Examples: Numbering 1-5, Bullet lists

Features: Custom numbering, bullet styles, indentation

üìä 6. Table Styles

Purpose: Formats tables with predefined layouts

Examples: Table Header, Table Contents

Features: Cell borders, background colors, text alignment

Working with Styles

Applying Styles

Step 1: Select Content

Place cursor in paragraph or select text to format

Step 2: Open Styles Panel

Press F11 or click Styles in sidebar

Step 3: Choose Style Category

Click on appropriate style category icon

Step 4: Apply Style

Double-click on desired style name

Creating Custom Styles

  1. Format text manually with desired appearance
  2. Select the formatted text
  3. Right-click in Styles panel
  4. Choose "New Style from Selection"
  5. Give it a meaningful name

Modifying Existing Styles

Right-click on any style name and select "Modify" to change fonts, colors, spacing, and other formatting options. All text using that style will automatically update.

Image-Text Relationship

Understanding Anchoring

Anchoring determines how images behave when text changes or when the document is edited. The anchor point controls the image's position relative to text elements.

üìÑ As Character

Behavior: Image sits on text baseline like a large character

Movement: Flows with text, no independent positioning

Use case: Inline icons, mathematical symbols

üîó To Character

Behavior: Image moves with specific character but can float

Movement: Follows anchor character when text reflows

Use case: Annotations, side notes

üìù To Paragraph

Behavior: Image stays with paragraph (default setting)

Movement: Moves when paragraph moves to new page

Use case: Most common for document images

üìÑ To Page

Behavior: Image fixed to specific page position

Movement: Stays in exact location regardless of text

Use case: Letterheads, watermarks, logos

üñºÔ∏è To Frame

Behavior: Image anchored to text frame or text box

Movement: Moves with the containing frame

Use case: Complex layouts, newsletters

Changing Anchor Type

  1. Select the image
  2. Right-click and choose "Anchor"
  3. Select desired anchor type from submenu
  4. Notice anchor symbol changes in document

Text Wrapping Options

Text wrapping controls how text flows around images and objects.

No Wrap

Text appears only above and below image

Page Wrap

Text flows around all sides of image

Optimal Wrap

Text wraps on side with more space

Left/Right Only

Text appears on specified side only

Through

Text can overlap transparent areas

In Background

Image behind text (watermark effect)

Setting Text Wrap

  1. Select image and right-click
  2. Choose "Wrap" from context menu
  3. Select desired wrapping option
  4. Adjust spacing if needed

Spacing and Margins

Control distance between image and surrounding text:

  • Before/After: Horizontal spacing on left and right
  • Above/Below: Vertical spacing on top and bottom
  • Synchronize: Use same value for all sides

Advanced Arrangement

Layer Management

Control stacking order when objects overlap:

  • Bring to Front: Move to top layer
  • Send to Back: Move to bottom layer
  • Bring Forward: Move up one layer
  • Send Backward: Move down one layer

Protection Settings

  2. Go to Options tab
  3. Check "Position" to lock position
  4. Check "Size" to lock dimensions

Templates are pre-designed document layouts that serve as starting points for new documents. They contain predefined styles, formatting, placeholder text, and structural elements that ensure consistency across similar documents.

üéØ Benefits of Using Templates

Table of Contents

What is a Table of Contents?

A Table of Contents (TOC) is an automatically generated list of document headings with corresponding page numbers. It provides readers with a quick overview of document structure and enables easy navigation to specific sections.

üìö Benefits of TOC

  • Navigation: Quick access to any section of the document
  • Overview: Shows document structure at a glance
  • Professional appearance: Adds polish to formal documents
  • Automatic updates: Page numbers update when content changes
  • Hyperlinks: Clickable links in digital documents

Prerequisites for TOC

Before creating a table of contents, your document must be properly structured with heading styles.

Using Heading Styles

Step 1: Apply Heading Styles

Select chapter/section titles and apply Heading 1, Heading 2, etc.

Step 2: Maintain Hierarchy

Use Heading 1 for main chapters, Heading 2 for sections, Heading 3 for subsections

Step 3: Verify Style Application

Check that all headings show correctly in Navigator (F5)

Heading Style Guidelines

  • Consistency: Apply heading styles uniformly throughout document
  • Hierarchy: Don't skip heading levels (e.g., Heading 1 directly to Heading 3)
  • Manual formatting: Avoid manual bold/size changes; modify heading styles instead
  • Outline levels: Each heading style has an associated outline level for TOC generation

Creating a Table of Contents

Basic TOC Creation

Step 1: Position Cursor

Place cursor where you want the TOC (usually after title page)

Step 2: Insert TOC

Insert > Table of Contents and Index > Table of Contents, Index or Bibliography

Step 3: Configure Settings

Choose title, protected against changes, and outline levels

Step 4: Click OK

Writer generates TOC based on heading styles in document

TOC Dialog Options

Type Tab

Title, type, protected status, evaluation level

Entries Tab

Structure, formatting, tab stops, hyperlinks

Styles Tab

Paragraph styles for each TOC level

Columns Tab

Multi-column TOC layout options

Background Tab

Background color or image for TOC

Customizing Table of Contents

Adjusting TOC Levels

Control which heading levels appear in TOC:

  • Evaluate up to level: Set maximum heading level (1-10)
  • Create from: Choose to use outline levels or specific styles
  • Additional styles: Include custom paragraph styles in TOC

Formatting TOC Entries

Customize appearance of TOC entries on Entries tab:

  • Structure line: Define entry format with tokens (E# = entry text, T = tab stop, # = page number)
  • Tab stops: Position and alignment of page numbers
  • Character styles: Apply formatting to specific tokens
  • Fill character: Dots, dashes, or other characters between text and page numbers

TOC Styles

Each TOC level uses a specific paragraph style:

  • Contents 1: Style for first-level headings in TOC
  • Contents 2: Style for second-level headings in TOC
  • Contents 3: Style for third-level headings in TOC
  • Modify styles: Change fonts, indentation, spacing for each level

Managing Table of Contents

Updating TOC

When document content changes, update the TOC:

Right-click Method

Right-click inside TOC > Update Index/Table

Menu Method

Tools > Update > All Indexes and Tables

Automatic Update

Enable in TOC properties for live updates

Editing TOC Properties

  1. Right-click anywhere in TOC
  2. Select "Edit Index/Table"
  3. Modify settings in dialog
  4. Click OK to apply changes

Deleting TOC

  1. Right-click anywhere in TOC
  2. Select "Delete Index/Table"
  3. Confirm deletion when prompted

Advanced TOC Features

Multiple Tables of Contents

Create different TOCs for different purposes:

  • Partial TOC: TOC for specific document sections
  • Table of Figures: List of images and captions
  • Table of Tables: List of tables in document
  • Custom TOC: Based on specific paragraph styles

Hyperlinked TOC

Create clickable TOC entries:

  1. In TOC dialog, go to Entries tab
  2. Click in structure line before "E#" token
  3. Click "Hyperlink" button (LS)
  4. Click after "E#" token and insert "Hyperlink end" (LE)
  5. TOC entries become clickable links to headings

TOC in Master Documents

For large documents split into subdocuments:

  • Master TOC: Includes headings from all subdocuments
  • Cross-references: Page numbers span entire master document
  • Automatic updates: Reflects changes in any subdocument
